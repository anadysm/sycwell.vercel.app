Citizen

APP & WEB SERVICE

The backdrop for filing the Citizen application is the number of damages to public facilities in the Sidoarjo Regency. The data gathered via this application is believed to be valuable for the government in evaluating the performance of public facilities that have been developed, thanks to the use of a web service system.

Technician-ACT

WEBSITE

Technician-ACT is a website that allows Telkom network customers to report a ticket with an abnormality status on the measurement system attenuation. Telkom staff no longer have to choose and sort reports thanks to this website. Because everything on the website has been subjected to measures by date and the current state of the network. Because the website does not require Telkom's intranet, it makes it easier for Telkom technicians to access user data that is experiencing an abnormality.
