Technician-ACT

WEBSITE

Technician-ACT is a website intended for report a ticket with an anomaly status on the measurement system attenuation, which is found in Telkom network users. With this website, Telkom employees no longer bother in choosing and sort reports. Because everything on the website have gone through measurements by date, and status current network condition. This website also helps make it easier for Telkom technicians to access user data experiencing an anomaly, because the website does not need a network Telkom's intranet.
